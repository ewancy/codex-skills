# 后台配置原型规范

## 当前后台页面优先

- 先查找当前后台截图、旧后台 demo、禅道附件或后台设计规范；找不到时向用户索要。
- 未获得用户明确授权前，不要把后台改成另一套布局或视觉风格。
- 新增配置项、表格列、弹窗或页签应沿用当前后台的顶栏、按钮、表单行、表格、弹窗和分页样式。

- 严格参考来源截图，不要自由换风格。
- 常见结构：顶部 `WEB开发`，右侧平台选择和用户，页面标题，筛选白块，操作按钮区，数据表格。
- 弹窗结构：标题栏、关闭按钮、页签、表单内容、底部提交/取消。
- 常见按钮：绿色主按钮、蓝色辅助按钮、白色线框按钮。
- 表单项按行展示，label 左侧，控件右侧。
- 字段联动必须真实：选择 A 时隐藏不适用字段，选择 B 时恢复。
- 不要放需求说明侧栏或说明卡片，除非用户明确要求。
